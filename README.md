# BasicOpenGLRenderer
Basic OpenGL 4.1 (MacOS only renderer)

# Build & Run (W/ CMake)
git clone https://github.com/ST-Programmer/BasicOpenGLRenderer/

cd BasicOpenGLRenderer

cd BasicOpenGLRenderer

mkdir build

cmake -B build -S .

cmake --build build

./build/3DOpenGL
